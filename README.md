# Productionization-of-ML-Systems-
A Streamlit-based ML app that predicts total travel cost using a trained SVR model. Users enter trip details like flight price, hotel cost, days, and travel distance to get a real-time prediction. Includes model scaling, feature engineering, and deployment-ready design.
